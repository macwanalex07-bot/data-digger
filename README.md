# Data Digger (SQL Project)

## Overview
Hands-on MySQL project for an e-commerce store using CRUD, clauses, operators, aggregates, PK/FK.

## Setup
1. Run `schema.sql` to create tables.
2. Run `sample_data.sql` to insert sample records.
3. Run `queries.sql` to test all tasks.

## Tables
- Customers
- Orders
- Products
- OrderDetails

## Features
- CRUD operations
- Filtering, sorting
- Aggregate functions (SUM, MAX, MIN, AVG)
- Joins for reporting
